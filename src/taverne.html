Description du projet

Ce cours niveau de jeu a été réalisé lors d’un cours de jeu vidéo. Pour sortir de la taverne, il faut trouver les 3 clés cachées à différents endroits et les ramener dans les serrures à côté de la porte.

Processus de création

Au début de mon processus de création, il me fallait un thème. J’ai cherché dans l’asset store de Unity pour des assets très simples en lowpoly, pour que je puisse facilement en créer de nouveaux au besoin.

Modélisation d’assets personnalisés

Par la suite, j’ai remarqué qu’il manquait quelques assets pour réaliser mon projet. J’ai donc utilisé mes compétences sur Blender pour modéliser la porte, le plancher/plafond, 2 types de murs et les serrures.

Création d'une flamme

J’ai aussi eu à utiliser le système de particules de Unity pour faire une animation de feu pour les torches. Il s’agit d’images 2D de feu qui rapetissent avec le temps jusqu’à disparaitre, ce qui donne l’illusion de flamme qui va assez bien avec le style lowpoly du reste de la scène.

Écriture d'un script

Dans ce niveau, il est possible de prendre tous les objets présents sur les tables (sauf les chandelles). Par contre, seules les clés peuvent ouvrir la porte (évidemment!). J'ai du écrire un script qui vérifie combien de serrures contiennent des clés. Si toutes les clés sont présentes, la porte s'ouvre.
